# Troubleshooting

Images should use the centralized deployed path:

```html
<img src="/silk-road/assets/images/dunhuang-dancers.jpg">
```

Checklist:

- The image file is inside `assets/images`
- The filename is lowercase
- The path starts with `/silk-road/assets/images/`
- The extension matches the real file exactly, such as `.jpg`, `.jpeg`, or `.png`
