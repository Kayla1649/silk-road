# Instructions for Using this Template

For reliable GitHub Pages deployment, keep project images in `assets/images` and reference them with the full deployed path:

```html
<img src="/silk-road/assets/images/hike-1.jpg">
```

Use lowercase filenames, no spaces, and exact extensions such as `.jpg`, `.jpeg`, and `.png`.

Good:

- `band-photo.jpg`
- `dunhuang-white.png`
- `tang-attire-picture.jpeg`

Avoid paths that depend on page location or nested folders. Do not use `images/filename.png`, `../images/filename.png`, or object-specific image folders in page content.
